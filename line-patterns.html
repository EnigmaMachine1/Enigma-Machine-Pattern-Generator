How to Use:

The lines category is the simplest of them. You simply divide the area into evenly spaced lines, and then fill lines with certain patterns generated. The first number of a lines code is 1. The second number of the lines code represents the direction and type of the lines that will be used to fill the space. Imagine each of these squares are the space you are supposed to fill, and here is an example of what the second number means for how you draw your lines in the space.

The third number of the lines code represents how many patterns will be repeated in sequence to fill the lines. It can be anywhere from 2-6 repeating patterns. Basically, if you had a 3 here, you would have 3 different patterns that you would use to fill the space in between the lines in sequence repeatedly, like so:

All of the next numbers with the little decimals next to them are the patterns that you will use to fill in the spaces. Below there is a list of the patterns that can be rolled, as well as an example of an actual code and its result. Note that 10.3 and 10.4 have a P denoting an empty space in the pattern. This empty space with a P means to roll a new pattern for that empty space. Don't worry, the generator shouldn't be able to generate another 10 for that pattern. When this happens, there will be a slash followed by the pattern to fill in here. For example, 10.3/1.2 would mean filling in the empty space in the 10.3 pattern with pattern 1.2.
