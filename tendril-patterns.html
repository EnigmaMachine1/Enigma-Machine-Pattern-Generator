How to Use:

The second number of the tendril pattern is always the pattern of tendril that you will have to draw. So I will list those patterns below.

Tendril Patterns:

Each line pattern is listed from left right (.1 being left most)

1. Straight

2. Curved

3. Centered

If your pattern roll started with a 3, the next number determines if your pattern is in the shape of a plus sign or an X. If it is a 1, it will be in the shape of a +, if it is a 2 it will be in the shape of an X.

The third number for patterns 1 and 2, and fourth for pattern 3 is the number that determines whether there will be a pattern inside the tendrils or not. If this number is a 1, the tendrils will be white like they are in the pattern pictures. If it is a 2, then the following number will be a pattern to fill the tendrils with. This is a line pattern, so use the line patterns from the code 1 are for reference.

After either the 1, or the pattern, there will be another number. This number represents what to do with the background. If it is a 1, the background will be black. If it is a two, the use the following pattern number from the line patterns to determine which pattern to fill in the background with. If it is a 3, the background will be an entire pattern. The pattern will be in the parentheses following the 3.
