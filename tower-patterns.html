How to Use:

The first number of the Towers pattern represents the direction the towers will point in. Note: if your second number is 2, just follow the direction and draw the second tower group directly opposite, pointing towards the tops of the first tower group.

The first number represents where the top of the towers is pointing. Interpret the first number as follows:

The next number represents the type of tower pattern you will be drawing. 1 is a single group of towers like this:

(5-7-1-3(4-2.2-1-1))

2 is two groups of opposing towers like this:

(5-3-2-3(4-2.2-1-1))

3 is towers branching out in a direction from one another like this:

(5-1-3-1)

As you can see in the examples above, just like tendrils the later number represent patterns to fill in the background with. If you rolled a 5-x-1 or 5-x-2, if the next number is a 1, you fill in the background black. If the next number is a 2, there will be a line pattern code to follow after the 2. If the next number is a 3, you will have an entire pattern in parentheses to fill the rest of the section with.

If you rolled a code that starts with 5-x-3, the next number will be either a 1 or a 2. If it is a 1, the background will be black, if it is a 2 the background will be the following line pattern number.
